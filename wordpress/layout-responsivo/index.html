Layout Responsivo

A cada dia nasce novos dispositivos: celulares, tablets, notebooks, desktops e novas resoluções (320×640, 1024×800, 4K, 5K).

Como isso seu website deve estar preparado para ser acessado por qualquer dispositivo e estar preparado é ter um layout responsivo que se adequa automaticamente independe da resolução do dispositivo que o acessou.
